Highlight Test Page

This page is for testing the search highlight functionality.

Test Content

This page contains various keywords to test the highlight functionality:

  • Whitesnake - A Stand in Bridger Western
  • The World - Time-stopping Stand
  • D4C - Dimension-hopping Stand
  • Killer Queen - Bomb-planting Stand
  • Corpse Part - Required to obtain Stands
  • Arrow Shard - Alternative Stand obtainment method

How to Test

Add ?highlight=world to the URL to see "world" highlighted.

Add ?highlight=stand to the URL to see "stand" highlighted.

Add ?highlight=corpse to the URL to see "corpse" highlighted.

Expected Behavior

When you add the highlight parameter, matching text should be:

  • Highlighted with yellow background (#ffd700)
  • Have dark text color (#2b1c11)
  • Have padding and border-radius
  • Pulse with animation 2 times
  • Automatically scroll into view
